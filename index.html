🎨 用 AI 畫 SVG & Canvas

不會寫 code 也沒關係 — 把需求丟給 AI，它就幫你畫好

🧠 怎麼玩？

你只需要做三件事：

  • 1. 想好你要畫什麼（一隻貓、一個圖表、一個 Logo…）
  • 2. 打開 Grok、Gemini、ChatGPT 或 Claude
  • 3. 把提示詞（Prompt）貼給它，它就幫你生出 HTML
  • 4. 把 AI 給你的 HTML 存成 .html 檔案，用瀏覽器打開就看得到了

👇 下面每一個例子，你都可以直接複製 prompt 去 AI 試試看。

動態粒子 (Canvas)

Canvas
🤖 複製這段 Prompt 給 AI Create an animated particle system using HTML5 Canvas and JavaScript. Draw 80 floating circles in different sizes and soft colors that drift upward slowly and wrap around. Dark background with a subtle glow effect on each particle. Add a gentle mouse interaction where particles slightly avoid the cursor. Size 400x300.
📄 Canvas JavaScript 動畫 
JavaScriptconst canvas = document.getElementById('c');
const ctx = canvas.getContext('2d');
const particles = [];

for (let i = 0; i < 80; i++) {
  particles.push({
    x: Math.random() * canvas.width,
    y: Math.random() * canvas.height,
    r: 2 + Math.random() * 6,
    speed: 0.2 + Math.random() * 0.6,
    color: `hsla(${200 + Math.random() * 100}, 80%, 70%, 0.6)`
  });
}

function animate() {
  ctx.fillStyle = 'rgba(10,10,30,0.15)';
  ctx.fillRect(0, 0, canvas.width, canvas.height);
  for (const p of particles) {
    p.y -= p.speed;
    if (p.y < -p.r) p.y = canvas.height + p.r;
    // mouse avoidance ...
    ctx.beginPath();
    ctx.arc(p.x, p.y, p.r, 0, Math.PI * 2);
    ctx.fillStyle = p.color;
    ctx.fill();
  }
  requestAnimationFrame(animate);
}
animate();

互動時鐘 (Canvas)

Canvas
🤖 複製這段 Prompt 給 AI Create an analog clock using HTML5 Canvas and JavaScript. Draw a circular face with hour markers (1-12), hour, minute, and second hands that update in real time. Use a dark theme with glowing neon-style hands. Smooth second hand movement. Size 300x300.
📄 Canvas JavaScript 時鐘 
JavaScriptconst c = document.getElementById('clock');
const ctx = c.getContext('2d');
const cx = 150, cy = 150, r = 130;

function drawClock() {
  const now = new Date();
  const h = now.getHours() % 12, m = now.getMinutes(), s = now.getSeconds();

  ctx.clearRect(0, 0, 300, 300);
  // face
  ctx.beginPath();
  ctx.arc(cx, cy, r, 0, Math.PI * 2);
  ctx.fillStyle = '#0f172a';
  ctx.fill();
  ctx.strokeStyle = '#334155';
  ctx.lineWidth = 4;
  ctx.stroke();

  // hour markers ...
  for (let i = 0; i < 12; i++) {
    const a = (i * 30 - 90) * Math.PI / 180;
    ctx.beginPath();
    ctx.moveTo(cx + Math.cos(a)*(r-18), cy + Math.sin(a)*(r-18));
    ctx.lineTo(cx + Math.cos(a)*(r-6), cy + Math.sin(a)*(r-6));
    ctx.strokeStyle = i === 0 ? '#fbbf24' : '#64748b';
    ctx.lineWidth = i === 0 ? 4 : 2;
    ctx.stroke();
  }

  // hands ...
  // hour hand
  const ha = ((h + m/60) * 30 - 90) * Math.PI / 180;
  ctx.beginPath();
  ctx.moveTo(cx, cy);
  ctx.lineTo(cx + Math.cos(ha)*50, cy + Math.sin(ha)*50);
  ctx.strokeStyle = '#60a5fa';
  ctx.lineWidth = 6;
  ctx.stroke();
  // minute hand ...
}
setInterval(drawClock, 1000);
drawClock();

💡 給 AI 下 prompt 的小技巧

  • 要具體：不要只說「畫一隻貓」，而是說「用 SVG 畫一隻橘色的貓，圓臉、尖耳朵、長尾巴，大小 400x400」
  • 指定格式：開頭直接說 用 SVG 或 用 HTML5 Canvas + JavaScript
  • 給尺寸：加上 尺寸 500x400 避免 AI 亂猜大小
  • 要顏色：說清楚顏色，例如 深藍色背景 + 金色文字
  • 要動畫：如果是 Canvas，說 加上動畫，粒子會慢慢往上飄
  • 一鍵複製：直接跟 AI 說 給我一份完整的 HTML 檔案，打開就能用

🤖 推薦 AI 工具

  • Grok (xAI) — 免費，擅長生成 SVG，回覆直接
  • Gemini (Google) — 免費，對 HTML/Canvas 支援好
  • ChatGPT (OpenAI) — 穩定，適合複雜動畫
  • Claude (Anthropic) — 程式碼品質高，結構清楚
